# 网上书店（BookMark）

## 项目说明

这是一个Java项目，采用前后端分离，实现网上书店的基本功能，比如：

* 用户登录注册
* 图书商品的查询和浏览
* 购物车系统
* 订单系统（可退货可取消）
* 用户基本信息管理
* ...（没想好，后面再改）

## 使用的框架

* SpringBoot
* Mybatis-Plus
* ...（没想好，后面再改）
